# CS-255 System Analysis and Design

## Overview
This repository contains coursework and project artifacts for CS-255: System Analysis and Design. The focus of this course is on analyzing business requirements and designing effective system solutions using industry-standard modeling techniques.

## Objectives
- Apply system analysis methodologies
- Develop UML diagrams to model system behavior and structure
- Translate business requirements into technical designs
- Understand software development lifecycle (SDLC) concepts

## Contents
- UML Diagrams (Use Case, Sequence, Activity, etc.)
- System design documents
- Requirements analysis
- Project deliverables

## Tools & Technologies
- Lucidchart (for UML diagrams)
- Microsoft Word / PDF (documentation)
- GitHub (version control and portfolio)
